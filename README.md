# Shopify Product CSV Generator

ブラウザ上でShopify用の商品CSVを生成できるGUIアプリケーションです。

## 機能

- ✅ ブラウザ上でGUIでCSV生成
- ✅ 1商品に対して複数枚の写真選択可能（なしも可能）
- ✅ 同時に複数の商品・バリエーションを作成・管理
- ✅ 各フィールドに説明表示（?アイコンでツールチップ）
- ✅ 固定値はデフォルト値として入力済み
- ✅ ドロップダウンリストで選択可能な項目
- ✅ CSVテンプレートの読み込み・編集・エクスポート

## 使い方

1. `index.html` をブラウザで開く
2. 「新規商品を追加」ボタンで商品を登録
3. 「CSVをダウンロード」ボタンでCSVを出力

## 画像について

ブラウザのセキュリティ制約により、**ローカルファイルシステムに直接アクセスしてフォルダにコピーすることはできません**。

以下の2つの方法で対応できます：

### 方法1: Data URLとしてCSVに埋め込み（現在の実装）
- アプリ内でアップロードした画像はData URL形式でCSVに保存されます
- Shopifyにアップロードする際は、画像を別途アップロードしてURLを置き換えてください

### 方法2: 手動で画像URLを指定
1. 画像を別途サーバー（ShopifyやCDN）にアップロード
2. 取得した画像URLをCSVの `Product image URL` 列に貼り付け

## フィールド説明

| フィールド | 説明 | デフォルト値 |
|-----------|------|-------------|
| Vendor | メーカー・ブランド名 | Your Brand |
| Status | 商品の公開状態 | active |
| Published | オンラインストアで公開 | TRUE |
| Charge tax | 消費税の有無 | TRUE |
| Requires shipping | 配送が必要か | TRUE |
| Inventory Policy | 売り切れ時の対応 | DENY |
| Fulfillment Service | 配送処理方法 | manual |

## バリエーション設定

1. Option1名・Option2名・Option3名でオプション種類を選択
2. 各「値」フィールドにカンマ区切りで値を入力
   - 例: `Small, Medium, Large`
   - 例: `Red, Green, Blue`
3. 自動的に組み合わせが生成され、SKUも自動生成されます

## CSVテンプレート読み込み

既存のShopifyエクスポートCSVを読み込んで編集できます：
1. 「テンプレートを読み込み」ボタンをクリック
2. CSVファイルを選択
3. 自動的に商品リストが復元されます
