# デプロイ手順

このアプリをインターネット上に公開し、チームメンバーに共有する手順です。

## 方法1: Netlify Drop（最も簡単・推奨）

1. ブラウザで https://app.netlify.com/drop を開く
2. 以下のファイルを含むフォルダをドラッグ&ドロップ：
   - `index.html`
   - `style.css`
   - `app.js`
   - `netlify.toml`
3. 自動的に公開URLが発行される（例：`https://xxx-yyy-zzz.netlify.app`）
4. URLをチームメンバーに共有

**パスワード**: `fullgram2026`
（一度入力するとブラウザに記憶されます）

## 方法2: Vercel（CLIでデプロイ）

1. ターミナルで以下を実行：
```bash
npm install -g vercel
vercel login
```
2. このフォルダで以下を実行：
```bash
vercel --prod
```
3. 公開URLが発行される

## 方法3: GitHub Pages

1. GitHubで新しいプライベートリポジトリを作成
2. このフォルダのファイルをプッシュ
3. Settings → Pages → Sourceを「main branch」に設定
4. `https://<username>.github.io/<repo-name>` でアクセス可能

---

**注意**: どの方法でも、アプリ内のパスワード認証（`fullgram2026`）が機能します。
ただし、ソースコード内のパスワードは簡易的なものなので、高度なセキュリティが必要な場合はNetlifyの「Site protection」機能（有料）を検討してください。
